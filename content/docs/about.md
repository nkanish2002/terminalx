# About TerminalFS

A terminal-style static-site framework. Interactive, type-a-command experience powered by Markdown.

## Mission

Make static sites feel like interactive terminals — fast, keyboard-driven, and familiar to developers.

## Design Principles

- **Markdown-first** — Content in `.md` files, real files, edit + commit
- **Configurable** — One `terminal.config.ts` controls everything
- **Framework, not site** — Engine is reusable; content is per-site
- **Zero runtime deps** — Vanilla HTML/CSS/JS SPA, build-time deps only

## Why Terminal?

Terminal UIs are:
- Fast to navigate (keyboard-only)
- Familiar to developers
- Compact (lots of info, small space)
- Fun to use
