# Dashboard Admin

Web-based dashboard for server management, process monitoring, and log viewing.

## Features

- Real-time process monitoring
- Log file viewer with search
- Service management (start/stop/restart)
- System resource metrics

## Tech Stack

- FastAPI backend
- Solara frontend (Textual TUI migration in progress)
- Docker containerization

## Access

Deploy to `0.0.0.0:9119` with `--insecure` flag (deprecated).

## Future

- Textual TUI migration
- Docker image updates
- OAuth auth support
